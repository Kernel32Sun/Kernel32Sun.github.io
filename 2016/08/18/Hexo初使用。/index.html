Hexo初使用。

By 岁月。

2016-08-18 Updated:2016-08-21

Contents

环境配置
安装Git，Mac默认有。
安装Node.js,官网下载。
安装过程略过，一直下一步就好。

安装Hexo
1）为博客创建一个目录并进入目录中。
2）安装Hexo，sudo npm install -g hexo
3）接着初始化，hexo init

至此安装工作全部完成。

4）可以生成静态页面了，hexo g（hexo generate）
5）启动服务，hexo server，之后可以在游览器中打开http://localhost:4000。

配置Github
1）建立Repository
2）建立与用户名相对应的仓库，仓库名必须为【your_user_name.github.io】
3）添加Ssh公钥，执行命令，生成密钥，ssh-keygen -t rsa -C “你的账户”，拷贝id_rsa.pub的全部内容，添加SSH公钥到『Account settings -> SSH Keys -> Add SSH Key』，执行命令ssl -T git@github.com，显示为：
Warning: Permanently added the RSA host key for IP address ‘192.30.253.112’ to the list of known hosts.
PTY allocation request failed on channel 0
Hi xxxx! You’ve successfully authenticated, but GitHub does not provide shell access.
Connection to github.com closed.则成功。
4）修改_confi.yml文件，在根目录下，到最下面修改成这样：
deploy:

type: git

repo: https://github.com/your_user_name/your_user_name.github.io.git

branch: master

5)执行命令npm install hexo-deployer-git –save，hexo deploy。
6）之后就可以在游览器访问了。

全部工作完成，第一次部署Hexo，记录一下。
