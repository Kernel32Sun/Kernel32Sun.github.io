华为机顶盒破解。

发表于 2016-12-13   |     |  

打开电视看电影，发现家里是智能电视了竟然，奈何装不了APP(无耻的运营商)，本着方便爸妈便宜自己的原则，遂剁之。

0X1 收集信息

机顶盒型号 ：华为EC6106V8
网络环境：光猫 + 路由
所需工具：STB管理工具，adb工具

0X2 破解

1）保持机顶盒跟电脑在同一局域网内，之后PING一下机顶盒通不通，如下图正常状态。

2）打开机顶盒设置-更多-远程维护连接-启用，如不启用STB工具将无法连接。
3）通过STB工具连接机顶盒，IP地址自己看一下吧，用户名：huawei 密码：.278aW，然后登陆，之后需要小小的配置一下机顶盒，主要开启Telnet服务和串口，之后就可以干好多好多的事情啦。
具体配置看图：


依次开启服务然后提交，切记。
4）安装APP及双网切换

首先 adb connect 192.168.x.x连接到所需设备，之后abd install XXX.apx，到此破解工作就此完成，机顶盒可以安装各类APP了，需要卸载时执行adb uninstall + 包名（注：包名不是应用名称，可将APK解包后，查看AndroidManifest.xml得到，如下所示）

双网切换不知道是不是为所在地区的问题，解决方法为连接两个路由器，这样看电视与看视频不冲突，不会卡顿。
